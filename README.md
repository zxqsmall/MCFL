# MCFL-master
The code for our paper **Multi-label Contrative Focal Loss for Pedestrian attribute recognition**. Applied a new loss function for PAR, we get the considerable performance based on the basic ResNet-50.
# Dependecies
- pytroch 1.4.0
- torchvision 0.5
- easydict 1.9
# Dataset
- RAP
- PA100K
- PETA
